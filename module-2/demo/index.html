Layer 1 — Frontend What the browser sees
What is the frontend? Everything the user sees and touches. HTML structures the page, CSS styles it, JavaScript makes it interactive. The frontend collects your input and sends it somewhere — but it can't store data or call AI on its own. It needs to talk to a backend.

Layer 2 — Backend What the server does
What is the backend? Code running on a server that the user never sees. It receives requests from the frontend, validates and processes them, talks to databases, calls external APIs (like Claude), and sends a response back. Without a backend, the frontend is just a static page.

Layer 3 — AI Backend What Claude does
What is an AI backend? A special backend that routes requests to a language model (like Claude) instead of — or in addition to — a traditional database. The backend formats your input into a prompt, sends it to the AI API, and returns the generated response to the frontend. This is the layer Claude Code builds for you.
